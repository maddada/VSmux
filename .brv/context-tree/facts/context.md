# Domain: facts

## Purpose

Stores project-level factual statements that are useful for quick recall across the repository.

## Scope

Included in this domain:

- Technology choices
- Configuration defaults
- Behavioral invariants
- Operational thresholds and constants

Excluded from this domain:

- Long-form architectural narratives better suited to architecture topics

## Ownership

Project knowledge base

## Usage

Use this domain for concrete repository facts that should be easy to query later.
